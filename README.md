## 🌟 Key Features

- **Modern Glassmorphism UI:** A sleek dark mode design system built with custom CSS variables, responsive typography, and glowing hover effects.
- **Interactive Hero Canvas:** Features a dynamic particle/nodes background rendered using HTML5 Canvas.
- **Animated Typing Effect:** Dynamic typewriter effect displaying key developer roles.
- **Floating Tech Badges:** CSS-animated floating programming language icons in the hero section.
- **About & Detailed Expertise:** Clear breakdown of technical skills across languages, frontend, backend, databases, AI/NLP, tools, and computer science fundamentals.
- **Internship Timeline:** Modern vertical timeline displaying professional experience (Ethara AI - Post-LLM Intern).
- **Featured Projects Grid:** Card layout displaying major engineering works (FilmoraX and OPTIMUS) with live demo and source code links.
- **Contact Integration:** A complete contact form and instant redirection buttons to WhatsApp (+91 7037366838), LinkedIn, and Email.

## 🚀 Local Setup

To run this project locally, follow these simple steps:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Sumant9761/Portfolio.git
   ```
2. **Navigate to the Directory:**
   ```bash
   cd Portfolio
   ```
3. **Open the Project:**
   Open `index.html` in any web browser, or use VS Code's "Live Server" extension for real-time previewing.
